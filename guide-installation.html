⚙️ GUIDE INSTALLATION

PORTAIL CAPTIF CYBER KIM — MIKROTIK HAP AC2

ÉTAPE 1 — Configurer les profils Hotspot

TERMINAL MIKROTIK

Ouvre Winbox → Terminal et colle ces commandes :

# ── Créer les profils de durée ── /ip hotspot user profile add name="Flash" session-timeout=00:15:00 rate-limit="2M/2M" shared-users=1 add name="Express" session-timeout=00:30:00 rate-limit="2M/2M" shared-users=1 add name="Starter" session-timeout=00:45:00 rate-limit="2M/2M" shared-users=1 add name="Standard" session-timeout=01:00:00 rate-limit="2M/2M" shared-users=1 add name="Confort" session-timeout=01:30:00 rate-limit="2M/2M" shared-users=1 add name="Plus" session-timeout=02:00:00 rate-limit="2M/2M" shared-users=1 add name="Max" session-timeout=02:30:00 rate-limit="2M/2M" shared-users=1 add name="Premium" session-timeout=03:00:00 rate-limit="2M/2M" shared-users=1 add name="Pro" session-timeout=03:30:00 rate-limit="2M/2M" shared-users=1 add name="Ultra" session-timeout=04:00:00 rate-limit="2M/2M" shared-users=1
# ── Activer la persistance des sessions (anti-coupure) ── /ip hotspot profile set [find] keepalive-timeout=00:02:00 /ip hotspot set [find] idle-timeout=00:05:00

ÉTAPE 2 — Uploader le portail personnalisé

FICHIERS À UPLOADER

Le portail MikroTik lit ses fichiers HTML depuis :

📁 flash/hotspot/ sur le routeur

2.1 — Via Winbox (méthode simple)

  • Ouvre Winbox → Files
  • Navigue dans le dossier hotspot
  • Glisse-dépose le fichier login.html
  • Glisse-dépose le fichier cyber-kim-logo.svg

2.2 — Via FTP (méthode alternative)

# Depuis ton PC, dans le terminal Windows (CMD) : ftp 192.168.17.1 # Login: admin / Mot de passe: (ton mot de passe MikroTik) cd flash/hotspot put login.html put cyber-kim-logo.svg bye

ÉTAPE 3 — Configurer le profil Hotspot par défaut

WINBOX → IP → HOTSPOT → SERVER PROFILES
# Dans le terminal MikroTik : /ip hotspot profile set [find] login-by=http-chap \ html-directory=hotspot \ http-cookie-lifetime=3d \ use-radius=no
✅ login-by=http-chap : authentification sécurisée par formulaire HTML

ÉTAPE 4 — Nommer le WiFi (SSID)

WINBOX → WIRELESS
# Renommer le réseau WiFi : /interface wireless set [find] ssid="CYBER KIM" # Optionnel : WiFi séparé pour les clients (bridge) /interface wireless set wlan1 ssid="CYBER KIM - WiFi"

ÉTAPE 5 — Vérification finale

CHECKLIST
  • ☐ Hotspot actif sur l'interface correcte (bridge/ether2)
  • ☐ Fichiers login.html et cyber-kim-logo.svg dans flash/hotspot/
  • ☐ Profils de durée créés (Flash, Express, Starter...)
  • ☐ Jetons générés via l'app Cyber Kim
  • ☐ Test depuis un téléphone : se connecter au WiFi "CYBER KIM"
  • ☐ Le portail s'ouvre automatiquement dans le navigateur
  • ☐ Entrer un code jeton → connexion réussie
⚠️ Si le portail ne s'affiche pas, assure-toi que le DNS est bien configuré sur 192.168.17.1 dans les paramètres Hotspot.

DÉPANNAGE
